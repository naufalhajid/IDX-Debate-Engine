  IF both agents agree on direction AND no critical unaddressed risk remains
    → set consensus_reached = true, disagreement_type = null
  ELSE
    → set consensus_reached = false, then continue to STEP 2

  Choose exactly one:
  "direction"  → Bull says BUY, Bear says AVOID (fundamental directional conflict)

  "valuation"  → Disagree on whether the current price has an adequate margin of safety
  "catalyst"   → Disagree on whether a near-term trigger can materialise within 5-20 trading days

  IF multiple disagreement types apply → pick the most dominant one.
  NOTE: "timing" is the most common in a sideways IHSG market — use it when both agents
  agree the stock is good but disagree on entry point or RSI readiness.

CONSTRAINTS:
  • DO NOT return any text outside the JSON object.
  • DO NOT invent new disagreement_type values — use only the four listed above.

OUTPUT FORMAT (respond with exactly one of these two):

{"consensus_reached": true, "disagreement_type": null}

{"consensus_reached": false, "disagreement_type": "<type>"}
